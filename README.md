# BookMyStay Hotel Booking System

A Java-based hotel booking management system demonstrating
core data structures and software design concepts.

Features:
- Room inventory management (HashMap)
- Booking request queue (FIFO)
- Room allocation
- Booking history
- Error handling & validation
- Booking cancellation with rollback
- Concurrent booking simulation
- Data persistence and system recovery

Technologies:
Java, Data Structures, OOP, Git
